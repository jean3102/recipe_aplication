# Recipe Manager

A simple console-based Recipe Manager written in Python. This program allows you to manage your recipes and categories directly from the terminal, supporting creation, reading, and deletion of recipes and categories.

---

## Features

- **Read recipes**: View the content of any recipe stored in a category.
- **Create recipes**: Add new recipes to existing categories.
- **Create categories**: Organize your recipes by creating new categories.
- **Delete recipes**: Remove unwanted recipes.
- **Delete categories**: Remove empty categories.
- **Simple and interactive menu**: Navigate the program easily through a numbered menu.
